Aplicacion web para la creacion de notas. El Front-End lo hice con html, css, bootstrap y javascript . EL BackEnd con Django y django rest framework para las API rest. Con este projecto me sumergi en el mundo del desarrollo web, utilizando por primera vez javascript para realizar las peticiones a mi servidor y mostrar de forma dinamica la informacion, profundice en mis conocimientos de CSS y HTMl.
En la parte del backend, (que es mi favorita) utilice sqlite como base de datos , modelo ORM para el modelado de mi BD y las querys , con django rest framework cree las APIs para la creacion, obtencion, edicion y eliminacion de notas
, utilice CSRF y tokens  de authenticacion para la seguridad con librerias como allauth para el registro de usuarios. Este proyecto contribuyo mucho para mi crecimiento personal

Aplicacion de Chat hecha en Python y tkinter para la interfaz grafica de usuario. Este proyecto me ayudo a aprender muchisimo acerca de como funciona las comunicaciones entre computadoras mediante la utilizacion de sockets y hilos. Aprendi un monton acerca de paralelismo , ya que para cada cliente del chat debia abrir un nuevo hilo para atenderlo, crear salas de chat, y permitir la comunicacion privada entre usuarios.
